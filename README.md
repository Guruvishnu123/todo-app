# Todo App

A simple and beautiful todo application built with HTML, CSS, and JavaScript.

## Features
- ✅ Add, delete, and complete tasks
- ✅ Filter todos (All, Active, Completed)
- ✅ Data persistence using LocalStorage
- ✅ Responsive design

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Use
1. Open index.html in your browser
2. Add tasks, mark complete, and delete as needed
3. Tasks are saved automatically

Open `index.html` in your browser!
